C'est quoi le GraphQL ?

GraphQL est un langage de requête et un environnement d'exécution côté serveur pour les interfaces de programmation d'application (API) qui s'attache à fournir aux clients uniquement les données qu'ils ont demandées, et rien de plus.

Qu'est ce que Hasura ?



Selon la documentation officielle, Hasura est un moteur d'API GraphQL. Il rend les données instantanément accessibles via une API GraphQL en temps réel, ce qui permet de créer et de livrer des applications et des API modernes rapidement. Hasura se connecte aux bases de données, serveurs REST, serveurs GraphQL et API tierces pour fournir une API GraphQL unifiée sur toutes sources de données. Les tables de base de données Postgres sont implétées directement dans la structure et les relations des requêtes et mutations GraphQL sans écrire de code.

Qu'est ce que permet Hasura ?

  • Fusionner des schémas distants : accéder à des schémas GraphQL personnalisés

  • Fonctionne avec des bases de données existantes et actives :pointer vers une base de données Postgres existante pour obtenir instantanément une API GraphQL prête à l'emploi
    • Effectuer des requêtes puissantes : filtrage intégré, pagination, recherche de modèles, insertion en masse, mise à jour, suppression de mutations

    • Temps réel : convertir n'importe quelle requête GraphQL en une requête en direct

    Hasura VS Prisma

    Hasura et Prisma fournissent une API GraphQL sur Postgres. Cependant, il existe une différence significative. Prisma en tant qu'API GraphQL sur votre base de données ne peut pas être utilisée directement par vos applications frontales.



    Hasura épargne le besoin d'écrire un backend CRUD et évolutif en temps réel sans interférer avec la façon dont vous écrivez votre logique métier.



    Conclusion

    Hasura est un outil complet, il permet la manipulation de données pour les moins experimentés en SQL. C'est un outil qui permet un bon intermédiaire entre la partie technique et métier d'une entreprise. La facilité d'utilisation dû à son aspect graphique reflète le potentiel qu'a Hasura pour devenir connu de tous.
